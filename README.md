# Team Task Manager

## Overview
The Team Task Manager is a project designed to assist teams in organizing, tracking, and managing tasks efficiently. It provides an intuitive interface for users to collaborate and ensure timely completion of projects.

## Features
- **Task Creation:** Users can create new tasks, assign them to team members, and set deadlines.
- **Collaboration:** Team members can comment on tasks, providing updates and discussing progress.
- **Status Tracking:** Users can update the status of tasks (e.g., To Do, In Progress, Done), allowing everyone to know the current state of work.
- **Prioritization:** Users can prioritize tasks to focus on whatâ€™s most important.
- **Responsive Design:** The application is designed to work on various devices, ensuring accessibility.

## Installation
To install the Team Task Manager locally, follow the steps below:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/bhanushekharyadav-glitch/Team-Task-Manager.git
   cd Team-Task-Manager
   ```

2. **Install Dependencies**:
   You need to have Node.js installed. Then run:
   ```bash
   npm install
   ```

## Setup
To set up the application:

1. **Environment Variables**:
   Create a `.env` file in the root directory and add the necessary environment variables. A sample file is provided as `.env.sample`.

2. **Run the Application**:
   Start the development server:
   ```bash
   npm start
   ```
   The application should be running on [http://localhost:3000](http://localhost:3000)

## Usage
Once the application is running:
1. Go to the home page where you can see all tasks.
2. Use the "Add Task" button to create new tasks.
3. Click on a task to view details and updates.
4. Update task status and add comments as necessary.
